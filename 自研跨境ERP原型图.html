经营主链路

对应需求中的商品 → Listing → 订单 → 库存 → 履约 → 售后 → 结算链路。

core loop
01
商品主数据 SPU、SKU、包装、申报信息为整个系统提供统一入口。
02
渠道与 Listing SKU 映射、站点模板、同步日志决定多市场运营质量。
03
订单与库存 订单三层 + 库存事务账，是一期可信度最高的部分。
04
履约与售后 面单、轨迹、退款、补发共同决定店铺健康度。
05
结算与利润 成本事件化后，才能真正回答“哪个 SKU 在赚钱”。
